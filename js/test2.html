First Box

In order to change the color of the table we use built-in bootstrap CSS classes: by adding/removing a class to the list of classes of the table element, we change the design of the table. 'table-dark' is a bootstrap class that defines a black table (see: the documentation).

In general you should avoid manipulating directly CSS properties (in this case you would have set the background color to black), because you can easily mess up the existing styles. And also because this is not a graphic design course....
